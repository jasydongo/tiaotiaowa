# 蛙一跳 · Jumping Frog 🐸

蛙一跳是一个类似微信「跳一跳」玩法的 WEB 小游戏。一只青蛙在森林的**木桩、蘑菇、石台、荷叶、树桩高塔**之间蓄力跳跃，落在平台中心可触发**完美连击**加倍得分。

纯原生实现：HTML5 Canvas 2D + 等距斜投影伪 3D，**无任何第三方依赖**，双击 `index.html` 即可运行。

**🌐 在线体验**：https://jasydongo.github.io/tiaotiaowa/

---

## 🎮 玩法

1. **按住**任意位置蓄力，**松开**起跳 —— 按得越久，跳得越远越高。
2. 落在下一个平台上 → **+1 分**，镜头前移，新平台不断刷新。
3. 落在平台**中心圈**内 → **完美落地**，连击越长奖励越高（完美奖励 = `2 × 连击数`：+2、+4、+6…）+ 金色粒子特效。
4. **玻璃平台**（每 30 次跳跃出现一次）→ 落上触发 **3 次自动连跳**，半透明辉光效果，连跳间隙无需操作。
5. 落空、掉下或跳过头 → 游戏结束。本地自动记录历史最高分。

## 🕹️ 操作

| 设备 | 蓄力 | 起跳 |
|------|------|------|
| 桌面 | 鼠标按住 / `空格` | 松开 |
| 手机 | 手指按住屏幕 | 松开 |

> 右上角两个按钮：
> - 🔊 切换音效与背景音乐（Web Audio 实时合成）
> - 🎯 显示/隐藏蓄力时的**落点辅助线**（含琥珀色靶心，轨迹与真实跳跃完全一致，落点预测精确）

## 🚀 运行

最简单 —— 双击 `index.html` 用浏览器打开即可。

或起一个本地静态服务器（可选，避免某些浏览器对本地文件的策略限制）：

```bash
# Python
python -m http.server 8000
# 然后访问 http://localhost:8000

# Node (http-server)
npx http-server -p 8000
```

## 🧩 技术要点

- **伪 3D**：等距斜投影（isometric），平台为顶面菱形 + 双侧面的立体方块，按 `worldX + worldY` 远近排序绘制（画家算法）。
- **物理**：蓄力曲线 → 抛物线跳跃 `z = 4·h·t·(1−t)`，落地挤压回弹形变。
- **镜头**：指数平滑跟随（`damp`，帧率无关），跳跃中追踪青蛙，静止时锁定当前平台。
- **判定**：青蛙与目标平台投影到屏幕后的距离对比，中心圈半径内即完美。
- **音效**：Web Audio API 振荡器合成（蓄力 / 跳跃 / 落地 / 完美 / 失败）+ C 大调音阶背景音乐，无需音频文件。
- **存档**：`localStorage` 持久化最高分（键名 `tiao_tiao_wa_best`）。
- **自适应**：DPR 高清渲染（上限 ×2），桌面 / 移动端布局与输入全适配。

## 🌲 平台与视觉

平台共 **6 种**随机样式：

| 类型 | 特点 |
|------|------|
| 🪵 木桩（stump） | 中等高度，顶面有年轮 |
| 🍄 蘑菇（mushroom） | 较矮，红伞白点，落地略软 |
| 🪨 石台（stone） | 矮，苔藓点缀，初始起点 |
| 🍃 荷叶（leaf） | 大而矮，放射状叶脉，落地略软 |
| 🗼 树桩高塔（tower） | 高而窄，顶面年轮少一圈 |
| 🔮 玻璃平台（glass） | 半透明辉光，每 30 次跳跃出现，落上触发 3 次自动连跳 |

主题为深绿森林夜景：层叠树影视差 + 漂浮萤火虫粒子（`lighter` 混合发光）+ 林冠月光光晕 + 地面雾气。
